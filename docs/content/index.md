---

lily is a set of Svelte 5 components you copy into your project and own. There is **one unified style** — no themes, no variants — extracted from the [levish.ac](https://levish.ac) design language.

Components are distributed like shadcn-svelte: a CLI fetches the source from the registry and drops it into your project, so you have full control over the code.

## Get started

Run the init command, then add the components you need:

```bash
npx lily init
npx lily add button
```
