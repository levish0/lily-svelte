lily is a calm, rounded set of Svelte 5 components — soft, minimal, and heavily rounded, with **one unified style**.

Components are distributed like shadcn-svelte: a CLI fetches the source from the registry and drops it into your project, so you own and can edit every file.

## Get started

Run the init command, then add the components you need:

```bash
npx lily-svelte@latest init
npx lily-svelte@latest add button
```
